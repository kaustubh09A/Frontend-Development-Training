# Frontend Development Training Project

## Overview
This project is created as part of Frontend Development Training. It covers HTML, CSS, Responsive Design, JavaScript Basics, DOM Manipulation, Events Handling, Arrays, Objects, and ES6 concepts.
---
## 🛠️ Technologies Used
* HTML5
* CSS3
* JavaScript (ES6)
---
