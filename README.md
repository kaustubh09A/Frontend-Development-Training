# Frontend Development Training Project
## Overview
This project is created as part of Frontend Development Training. It covers HTML, CSS, Responsive Design, JavaScript Basics, DOM Manipulation, and Form Validation.

## Technologies Used
* HTML5
* CSS3
* JavaScript
## Day 1 - HTML Fundamentals

### Topics Covered
* Basic HTML Structure
* Headings and Paragraphs
* Images and Links
* Forms
* Tables
* Lists
* Sections and Footer

### Features Implemented
* About Section
* Registration Form
* Student Details Table
* Programming Languages List
* Contact Section

## Day 2 - CSS Fundamentals
### Topics Covered
* CSS Selectors
* Box Model
* Flexbox
* Responsive Design
* Media Queries

### Features Implemented
* Navigation Bar
* Service Cards
* Hover Effects
* Responsive Layout
* Responsive Images
* Card Design using Box Model

## Day 3 - JavaScript Basics
### Topics Covered
* Variables and Data Types
* Functions
* Loops
* Conditions
* DOM Manipulation
* Form Validation

### Features Implemented
* Addition of Two Numbers
* Even/Odd Number Check
* Largest Number Among Three Numbers
* Simple Calculator
* Greeting Function
* Addition and Subtraction Function
* Multiplication Table
* For Loop, While Loop, Do-While Loop
* If, If-Else and Switch Case

### DOM Manipulation
* Change Text Content Dynamically
* Change Background Color on Button Click
* Show and Hide Elements
* Update Image Dynamically

### Form Validation
* Validate Name Field
* Validate Email Format
* Display Error Messages using JavaScript
